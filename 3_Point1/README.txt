Скетч для точки
1. Описание пинов:
  int pinRedTeam = 6;  //digital порт реле красной ленты
  int pinBlueTeam = 7; //digital порт реле синей ленты
  int pinIk = 8;       //digital порт инфракрасного приемника
  int pinSound = 9;    //digital порт пищалки (плюсовой контакт)

2. Режимы работы точки:
  - при запуске активируется включение цвета команды поочередно через каждые 5 секунд
  - первой команде необходимо выстрелить в точку, когда будет гореть свет, соответствующий цвету команды
  - дефолтное время срабатывания точки 10 минут.
3. Настройка и запуск аптечки:
  - для настройки времени срабатывания точки:
    1) нажать на кнопку menu. После нажатия, прозвучит одиночный звуковой сигнал, необходимо задать интервал срабатывания точки:
      1 - 5  минут
      2 - 10 минут
      3 - 15 минут
      4 - 20 минут
